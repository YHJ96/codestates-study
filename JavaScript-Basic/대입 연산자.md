# 대입 연산자

계산식을 간략히 나타내고 싶을 때 사용한다.

| 구문 | 의미 |
|:-----|:----|
| x += y | x = x + y |
| x -= y | x = x - y |
| x *= y | x = x * y |
| x /= y | x = x / y |
| x %= y | x = x % y |
| x **= y | x = x ** y |

```javascript
let num1 = 10;
let num2 = 20;
// num1 = num1 + num2 같은 의미
num1 += num2;
// 결과 : 30
console.log(num1)
```
