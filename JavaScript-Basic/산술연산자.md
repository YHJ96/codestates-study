# 산술연산자

## 이항 산술 연산자

| 구문 | 의미  |
|:----|:-------|
| +   | 덧셈   |
| -   | 뺄셈   |
| *   | 곱셈   |
| /   | 나눗셈 |
| %   | 나머지 |
| **  | 제곱   | 

```javascript
// 결과 : 30
console.log(10 + 20);
// 결과 : 70
console.log(100 - 30);
// 결과 : 300
console.log(100 * 3);
// 결과 : 100
console.log(500 / 5);
// 결과 : 1
console.log(101 % 2);
// 결과 : 16
console.log(2 ** 4);
```
