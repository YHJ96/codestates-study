# 스코프
식별자 접근 규칙에 따른 유효 범위
1) 안쪽 스코프에서 바깥쪽 스코프로 접근할 수 있지만 반대는 불가능하다.
2)스코프는 중첩이 가능하다.
3)전역 스코프와 지역 스코프


3-1) 전역 스코프 (global)

3-2) 지역 스코프 (Local Scope)

-함수 스코프
```javascript
var a = 1; // 전역 스코프
function print() { // 지역(함수) 스코프
 var a = 111;
 console.log(a);
}
print(); // 111
console.log(a); // 1
```

-블록 스코프 {}
```javascript
{
let a = 'aaa'
const b = 'bbb'
}
```
// 화살표함수는 블록 스콥흐다
