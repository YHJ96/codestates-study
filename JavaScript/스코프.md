# 스코프


## 📌 스코프(scope)란
> 변수 이름, 함수 이름, 클래스 이름과 같은 식별자가
 본인이 선언된 위치에 따라 다른 코드에서 자신이 참조될 수 있을지 없을지 결정되는 것


💻 
```js
function add(x, y) {
console.log(x, y);
return x + y;
}

add(3, 6);

console.log(x, y); //변수 x , y 가 선언된 위치에 따라 유효 범위가 정해진다.
//ReferenceError: x is not defined blah blah
```
